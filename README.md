# Basketball Quiz Game

## Description
This is a simple Java quiz game that tests knowledge of basketball terms.

- Score tracking
- Final feedback

## How to Run
- Compile the program
- Run the main class

## Roles (Solo Project)
- Menu: welcome screen and instructions
- Logic: questions and scoring system
- Output: results and feedback
